Исследование покупательской активности   

Описание проекта  

Интернет-магазин хочет оптимизировать маркетинговые затраты. Задача — построить модель, которая предскажет снижение активности клиентов в ближайшие 3 месяца. На основе прогноза необходимо выделить сегмент «рисковых» клиентов и предложить для них персонализированные предложения.  

Данные  

В работе использовались четыре группы признаков:  

Продуктовое поведение: данные о категориях и выручке.  

Покупательское поведение: частота визитов, акционные покупки.Поведение на сайте: время сессий, просмотры страниц.
Коммуникация: разрешение на рассылки.  

Стек технологий  

Python (Pandas, NumPy, Matplotlib, Seaborn)Scikit-learn (Pipeline, ColumnTransformer, RandomizedSearchCV)  

Модели: DecisionTreeClassifier, KNeighborsClassifier, LogisticRegression, SVC.  

Метрики: ROC-AUC.  

Ключевые этапы работы  

Предобработка данных: работа с пропусками, приведение типов, объединение таблиц.  

EDA: исследовательский анализ распределений и корреляций (включая анализ коэффициента $\phi_k$).  

Моделирование: создание автоматизированных пайплайнов для обработки категориальных и численных признаков.  

Подбор гиперпараметров: использование RandomizedSearchCV для поиска лучшей комбинации моделей и средств масштабирования.  

Анализ прибыли: оценка выручки клиентов и выделение целевого сегмента.  

Итоги исследования:  

Лучшая модель - SVC;  

Метрика ROC-AUC на кросс-валидации: 0.899
